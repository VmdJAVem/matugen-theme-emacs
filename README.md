# matugen-theme.el

A dynamic theme switcher for Emacs that synchronizes your Emacs colors with your wallpaper using [Matugen](https://github.com/InioX/matugen) and `modus-themes`.

Whenever you change your wallpaper using Matugen on your system (great for Wayland setups like Niri, Hyprland, etc.), this Emacs package automatically detects the color changes and live-reloads your theme without requiring an Emacs restart. 

It fully implements the **Material You** palette generated by Matugen directly on top of `modus-themes` (both `modus-vivendi` and `modus-operandi`!).

## Requirements
- Emacs 28.1+
- `modus-themes` package installed
- [Matugen](https://github.com/InioX/matugen) installed on your system

## 1. Configure Matugen
Matugen needs to export its generated color palettes into a JSON file that Emacs can read.

Create a template file, for instance at `~/.config/matugen/templates/colors.json`, containing:
```json
{
  "light": {
    "primary": "{{colors.primary.light.hex}}",
    "on_primary": "{{colors.on_primary.light.hex}}",
    "secondary": "{{colors.secondary.light.hex}}",
    "tertiary": "{{colors.tertiary.light.hex}}",
    "error": "{{colors.error.light.hex}}",
    "background": "{{colors.background.light.hex}}",
    "on_background": "{{colors.on_background.light.hex}}",
    "surface": "{{colors.surface.light.hex}}",
    "surface_variant": "{{colors.surface_variant.light.hex}}"
  },
  "dark": {
    "primary": "{{colors.primary.dark.hex}}",
    "on_primary": "{{colors.on_primary.dark.hex}}",
    "secondary": "{{colors.secondary.dark.hex}}",
    "tertiary": "{{colors.tertiary.dark.hex}}",
    "error": "{{colors.error.dark.hex}}",
    "background": "{{colors.background.dark.hex}}",
    "on_background": "{{colors.on_background.dark.hex}}",
    "surface": "{{colors.surface.dark.hex}}",
    "surface_variant": "{{colors.surface_variant.dark.hex}}"
  }
}
```

Then, tell Matugen to use this template by adding this to your `~/.config/matugen/config.toml`:
```toml
[templates.emacs-colors]
input_path = "~/.config/matugen/templates/colors.json"
output_path = "~/.cache/matugen-colors.json"
```

## 2. Install in Emacs

### Using Straight.el (Doom Emacs)
Add this to your `packages.el`:
```elisp
(package! matugen-theme
  :recipe (:host github :repo "diegoveraniego/matugen-theme-emacs"))
```

### Configuration (Doom Emacs)
In your `config.el`:
```elisp
(use-package! matugen-theme
  :defer nil
  :config
  ;; Starts the file watcher so Emacs reloads when Matugen writes the JSON
  (matugen-theme-mode 1))
```

## Usage

If `matugen-theme-mode` is enabled, your Emacs theme will update automatically when you run Matugen.

The package detects if your base theme is dark or light (`modus-vivendi` or `modus-operandi`) and loads the appropriate color palette.

You can also force a specific variant interactively:
- `M-x matugen-theme-load-dark`
- `M-x matugen-theme-load-light`
